Stair Climber 🏃‍♂️💨

疑似触覚（Pseudo-Haptics）と自己帰属感を利用した「身体的重量感」の検証アプリ

概要

Stair Climber は、キーボードの「上矢印キー」のみを使って、無限に続く階段を登り続けるインタラクションプログラムです。

スタミナが減るにつれてキャラクターの動きが重くなり、入力しているのに思うように進まない感覚が生まれます。
この「操作」と「画面上の反応」のズレによって、実際には触覚デバイスを使っていないにもかかわらず、指先や身体に重さを感じるような 疑似触覚（Pseudo-Haptics） の体験を目指しました。

単なるゲーム的なスタミナ表現ではなく、視覚フィードバックと操作感の関係から、デジタル空間に身体性を感じさせることを目的とした作品です。

制作背景

本作品は、大学の「インタラクションデザイン」の講義における 感触体験インタラクションデザイン をテーマに制作しました。

一般的なUXデザインでは、操作のしやすさや心地よさが重視されます。
一方で本作品では、あえて 操作しづらさ・もどかしさ・身体的な重さ を設計することで、ユーザーに「疲労」や「重量感」を感じさせる体験を目指しました。

特に以下の2つの考え方をもとに設計しています。

1. 疑似触覚（Pseudo-Haptics）

疑似触覚とは、視覚情報の変化によって、実際には存在しない触覚や抵抗感を感じさせる表現手法です。

本作品では、ユーザーのキー入力に対して、キャラクターの反応速度を徐々に遅くすることで、

* 指が重い
* 体が思うように動かない
* 階段を登るのがしんどい

といった感覚を視覚的に生み出しています。

2. 自己帰属感（Sense of Agency）

自己帰属感とは、「この動きは自分が操作している」と感じる感覚です。

本作品では、キャラクターの歩行アニメーションや身体バランスを丁寧に設計することで、ユーザーが画面内のキャラクターに自分の身体感覚を重ねやすくなるようにしました。

その結果、キャラクターの動きが遅くなったときに、単なる処理落ちやバグではなく、
自分自身のスタミナが切れて身体が重くなっているような感覚 として受け取られることを狙っています。

主な機能

階段を登るシンプルな操作

操作は「上矢印キー」のみです。
キーを押し続けることでキャラクターが階段を登り、登った段数がカウントされます。

スタミナによる動きの変化

スタミナが減少すると、キャラクターの動きが徐々に重くなります。

* 移動速度が低下
* 足の上がり方が小さくなる
* 体の沈み込みが大きくなる
* 左右のふらつきが増える
* 膝や肩の震えが発生する

これにより、視覚的に「疲れている」「限界が近い」ことが伝わるようにしています。

歩行アニメーション

キャラクターは左右の足を交互に動かしながら階段を登ります。
足の動きや体のバランスを調整することで、単なる図形ではなく、身体を持った存在として見えるように工夫しました。

リアルタイムUI

画面上には、現在の状態を示すUIを表示しています。

* スタミナゲージ
* 登った段数
* スタミナ低下時の警告表示

スタミナゲージは、残量に応じて 緑 → 黄 → 赤 に変化します。
また、スタミナが限界に近づくと "EXHAUSTED!" の警告を点滅表示し、視覚的な緊張感を高めています。

⸻

工夫した点

操作と視覚フィードバックのズレ

本作品の中心となる工夫は、入力に対する反応を意図的に遅らせる点です。

ユーザーは同じようにキーを押しているにもかかわらず、スタミナが減るにつれてキャラクターの動きが鈍くなります。
このズレによって、物理的な触覚デバイスを使わずに「重さ」や「抵抗感」を感じさせることを目指しました。

疲労感の誇張表現

疲労を単なるスピード低下だけで表すのではなく、身体のさまざまな変化として表現しました。

* 重心が下がる
* 足が上がらなくなる
* 体が左右に揺れる
* 肩が上下して呼吸が荒く見える
* 限界時に膝が震える

これにより、ユーザーがキャラクターの疲労をより直感的に感じられるようにしています。

デモしやすい入力設計

入力には文字キーではなく、上矢印キーを使用しました。

Processingの実行中にフォーカスが外れた場合でも、文字がエディタに大量入力される事故を防ぐことができます。
展示や発表の場でも安定して操作できるようにした点も工夫の一つです。

シルエットの調整

疲労表現を強くするために体を大きく前傾させると、キャラクターの見た目が崩れてしまいます。
そのため、姿勢の傾きは控えめにし、代わりに沈み込みや揺れを強調することで、見た目のバランスと疲労感の両立を図りました。

実行方法

1. Processing IDEをインストールする
2. このリポジトリをダウンロードまたはクローンする
3. .pde ファイルをProcessingで開く
4. 実行ボタンを押す
5. 上矢印キーを押して階段を登る

上矢印キー	階段を登る
