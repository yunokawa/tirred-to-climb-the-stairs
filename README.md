# tirred-to-climb-the-stairs
本作品は、キーボードの「上矢印キー」のみを用いて階段を登る、擬似触覚（Pseudo-Haptics）の実験的インタラクションデザインである。物理的な振動や重力フィードバックがないデジタル環境において、視覚情報と操作感の「ズレ」を利用し、ユーザーの脳内に身体的な「重み」や「疲労」を錯覚させる。 独自の歩行ロジックと「蹴上げ1：足2：身長5」の身体比率により画面内のキャラクターへの「自己帰属感」を高め、スタミナ減少に伴う移動速度の減速（指数関数によるタメ）を指先への抵抗感として知覚させる。さらに、疲労時の激しい呼吸、重力による体の沈み込み、足が上がらず階段の角をかすめる「つま先の引っかかり」などの誇張されたアニメーションを組み合わせ、デジタル空間における身体的苦痛の共有を可能にした。
